# Olist Brazilian E-Commerce — Sales Performance Case Study

A real-world end-to-end data analysis project using the public
[Olist dataset](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
from Kaggle (100,000+ orders, 9 relational tables).

## Tools
Python · Pandas · NumPy · Matplotlib · Seaborn · SciPy · Jupyter Notebook

## Dataset Setup
1. Download the dataset from the Kaggle link above
2. Unzip and place all 9 CSVs in the `/data` folder

## Project Structure
- `data/` — raw CSV files (not tracked in git)
- `notebooks/` — Jupyter notebooks for each sprint
- `reports/` — final case study outputs

## Sprints
- Sprint 1: Data Foundation (loading, cleaning, merging, EDA)
- Sprint 2: Deep Analysis & Statistics
- Sprint 3: Visualizations & Final Report
